# Task prompt template

Use this format when asking Phoney to write or edit something.

## Required
- **Who it’s for:** Sam / Rizom
- **Task:** what you want written or edited
- **Format:** post, bio, case study, note, newsletter, homepage copy, etc.
- **Audience:** who it’s for
- **Goal:** what the piece should do

## Helpful
- **Tone:** e.g. sharp, warm, institutional, playful, restrained
- **Reference material:** paste or attach any source docs
- **Constraints:** length, words to avoid, must-include points
- **Deadline / urgency:** if relevant

## Example
For Rizom:
Write a homepage positioning paragraph for agency clients.
Audience: potential clients.
Goal: explain what Rizom does in a crisp, credible way.
Tone: smart, confident, not hypey.
Reference: attached positioning notes.
